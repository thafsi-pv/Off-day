# OffDay Frontend

React 18 frontend for the OffDay roster management system.

## Setup

1. Install dependencies:
```bash
npm install
```

## Development

Run the frontend in development mode:
```bash
npm run dev
```

The frontend will run on `http://localhost:5173` by default and will proxy API requests to the backend at `http://localhost:3000`.

## Build

Build the production version:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

## Features

- User authentication (login/register)
- Leave request management for users
- Admin dashboard for managing leaves and users
- Configuration management
- Dark mode support

## Tech Stack

- React 18
- TypeScript
- Vite
- React Query
- Axios
- Tailwind CSS
- React Hot Toast
